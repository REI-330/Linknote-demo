## 主要视觉变化

### 1. 双色主题系统

- **白天模式（绫波丽）**
  - 背景 `#F2F5F8` 冷调月白
  - 主按钮/强调色：深灰钢蓝 `#6B7E99` 家族
  - 辅助色：`#C5D9F3` 冰雾浅蓝

- **黑夜模式（冬马和纱）**
  - 背景 `#12121A` 墨色冷黑
  - 主强调色：`#2A5CAA` 钴冰瞳蓝
  - 辅助色：`#283B61` 暗调藏青

### 2. 去除开发者文案

| 位置 | 去除内容 |
|------|----------|
| 日报页头部 | "首页只保留日报卡片流。点开任意条目后，再进入单条分析工作台。" |
| 分析设置卡片 | "设置页唯一来源" 徽章；长段 provider/model 挑选机制说明 |
| 原文片段空状态 | "这里会优先展示平台字幕..." 等解释性文字 |
| 下载配置底部 | 自动整理/开机自启/回看天数的多段机制说明 |

### 3. 单条分析页等待动画

- 新增 `AnalysisProgress` 四步进度条：
  1. 准备就绪
  2. 下载与转写
  3. AI 分析生成
  4. 生成完成
- 当前步骤高亮 + 脉冲动画
- 顶部旋转加载图标，底部展示后端返回的 `detail` 文本

### 4. 信息层级与间距

- 所有 Card 内边距统一收紧（`p-5` / `p-4`），减少臃肿感
- 多栏布局（笔记 / 原文 / 问答）间距从 `gap-4` 提升到 `gap-5`
- 三栏同时开启时最大宽度放宽至 `1400px`，更不易拥挤
- 输入框统一高度 `h-10`，与按钮对齐，减少视觉跳动

### 5. VideoBanner 去玻璃拟态

- 移除 `blur-xl` 与 `bg-white/15` 等玻璃效果
- 改用纯色渐变：
  - Light: `#6B7E99` → `#8FA3BD`
  - Dark: `#283B61` → `#2A5CAA`
- 封面图仅保留极低透明度作为纹理，不再喧宾夺主

## 取舍说明

- **没有引入新依赖**：shadcn/ui 基础组件能力已足够，未新增任何图标库或动画库，保持 bundle 体积不变。
- **没有自定义字体文件**：沿用 Inter（项目中已具备），通过提升字重对比（标题 `font-weight: 800`、 tighter `letter-spacing: -0.03em`）来模拟 X 平台的紧凑高对比感，而非引入网络字体。
- **保留语义色**：success（emerald）、warning（amber）等状态色未强行改成主题色，避免破坏用户对错/成功的直觉认知，仅在背景、文字、边框层应用新主题。
- **进度步骤是推断式**：由于后端 `stage` 字符串可能扩展，当前采用关键字匹配（`transcrib` / `analyz` 等）映射到 4 步。未来新增 stage 会自动回退到"准备就绪"步，不会报错。
